 What is Bankovia? Early Alpha

Bankovia is the economy bot developed by the Library of Code Community. This bot is only operated and overseen by the Library of Code Engineering Team. It is developed primarily by the community. The bot developers have administrator permissions over the bot. If you would like to report an issue, you may contact one of them.


 Bot Administrators

The bot administrators are those who operate the bot. You can basically consider them as official representatives of the Library of Code sp-us.
